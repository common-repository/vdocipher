How to customise the player with v2 version?

There is a custom player page in your vdocipher dashboard. Go there to make a custom player id of your own. Custom player can be used to change the looks of the video player when player version is set to v2.

A player is a collection of configuration options. These configuration options define how the video player is rendered. These options can toggle individual controls from the player, it can change the behaviour and set parameters for specific components. It can also let you change the general looks and design of the player.

Make sure to read the documentation available on the above page to learn how this work.
